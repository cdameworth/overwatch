Place AWS SVG icons in this directory for use in the dependency map visualization.

Recommended filenames:
- lambda.svg         (AWS Lambda)
- apigateway.svg    (API Gateway)
- iamrole.svg       (IAM Role)

You can download official AWS architecture icons from:
https://aws.amazon.com/architecture/icons/

Or use open-source icon sets such as:
https://github.com/awslabs/aws-icons-for-plantuml/tree/main/dist

Place the SVG files here and ensure the filenames match the mapping in app.js.
